The group.transfer_rate_limit file

In this file you can limit the transfer rate limit for a group of hosts. Each host (alias) may only be in one group. The syntax of this file is as follows: 

    [<group identifier>]
    members=<member 1>,<member 2>, ... <member n>
    limit=<limit in kilobytes per second>

There is no limit in the number of groups that can be specified, only make sure that a host is not in more then one group. A second limit for a host can still be specified in the HOST_CONFIG file or via the edit_hc dialog. If a host has two limits, always the lower value will take effect.
