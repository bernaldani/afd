AFD Configuration

MAX_COPIED_FILES If your system receives huge number of small files, set this value accordingly to a high value only if the system is fast enough to handle this load. If noti, set this to a low value, so AFD will start as early as possible with the distribution. This setting will effect all DIR_CONFIG values, unless max files is set under [dir options].
