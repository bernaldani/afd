Downloading AFD

For the graphical user interface AFD requires at least X11R5 and Motif 1.2.x (or higher). The AFD has so far been ported to the following systems:

  • Linux 1.3.x  -  2.6.x
  • Solaris 2.x
  • HP-UX 10.x  -  11.x
  • IRIX 5.3  6.x
  • AIX 4.3
  • MacOS 10.3
  • FTX 3.0.x  3.2.x
  • SCO OpenServer Release 5

It is distributed under the GNU General Public License - see the accompanying COPYING file for more details.

Please read the INSTALL file in the src directory, before compiling.

  • Download via anonymous ftp.
  • Latest stable release is 1.2.23 (1050 KB), source code only. To see what has been changed read Changes. If you upgrade from version 1.1.x please read and follow the instruction in this text file.
  • Precompiled binaries are also available via anonymous ftp.
  • Documentation (519 KB) is mostly done in HTML.
  • A sample AFD configuration can be found here (3 KB).
  • Todo list since the first Version of the AFD.

There is a mailing list at gafd-users@lists.sourceforge.net, for more information on how to register see gafd-users. An archive of this list can be found here.
